== Description ==

Post/Page specific Custom JavaScript will allow you to add your custom JS to specific posts or/and pages. It will give you special area in the post/page edit field to attach your JavaScript. It will also let you decide if this JS has to be added in multi-page/post view (like post archive) or only in a single view.

You will see additional meta box below text input area on posts and pages screen. You may add whatever javascript you want. You may choose if you want to add this JS code on certain single page/post or on pages like archive etc.

== Installation ==

= Automatic installation =

Automatic installation is the easiest. Simply log in to your WordPress admin panel, go to the Plugins menu and click Add New.

In the search field type Post/Page specific Custom JavaScript and click Search Plugins. Then you can install it by simply clicking &#8220;Install Now&#8221; button.

= Manual installation =

1. Upload `post-page-specific-custom-javascript` directory to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. That's all folks, have fun :)

== Requirements ==

This plugin requires at least WordPress <b>5.0</b> (never forget to update your WP installation!) and <b>PHP 5.6</b> installed on your server.
